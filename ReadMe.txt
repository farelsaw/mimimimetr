Мимимиметр написан на Spring Boot с использованием Maven, Thymeleaf и MySQL.

- Дамп БД catsdb со структурой и данными хранятся в папке DB\catsdb.sql;
- Папка web-приложения: mimimeter.

Начальная страница мимимиметра - localhost:8080, далее при нажатии "Запустить" запускается сам опрос на странице localhost:8080/start.
По завершению опроса из 5 пар котиков происходит переход на страницу топа котиков localhost:8080/top. 
При повторной попытке в рамках сессии запустить опрос происходит переброс на страницу топа котиков, так как опрос в рамках сессии уже пройден.
